# Digital Product Store API

Backend REST API for managing digital products such as ebooks, online courses, and software licenses.

## Features
- Product management
- Customer management
- Order management
- License management
- PostgreSQL integration
- GORM migrations

## Tech Stack
- Go
- Gin
- PostgreSQL
- GORM

## Endpoints

### Products
- GET /products
- GET /products/:id
- POST /products
- PUT /products/:id
- DELETE /products/:id

### Customers
- GET /customers
- POST /customers

### Orders
- GET /orders
- POST /orders

### Licenses
- GET /licenses
- POST /licenses
